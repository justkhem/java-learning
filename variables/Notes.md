# Variables and Data Types Notes

# Variable is a container that stores a value.

## Data Types I have learned so far:

- **String** - it stores text like "Khem"
- **int** - it stores whole numbers e.g. 25
- **double** - it stores decimal numbers e.g. 23.5
- **boolean** - it stores true or false
- **char** - it sstores a single character e.g. 'A'
- **float** - it also stores decimal (uses less memory & ends with the letter 'f' e.g. 22.23f)

## Things to remember:
- Always end statements with ;
- String uses double quotes ""
- char uses single quotes ''
- You can't redeclare a variable with its type
